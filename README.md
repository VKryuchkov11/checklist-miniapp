# 🚀 Checklist Mini App

Ultra-mega checklist inside Telegram.

Мини-приложение чек-листа прямо внутри Telegram.

---

# 🇬🇧 English

## ✨ Features

✅ Create tasks
📂 Task categories
✏️ Edit tasks
🗑 Delete tasks
📊 Progress tracking
🔐 Telegram user authorization
☁️ Backend server with FastAPI
🗄 SQLite database
🔔 Task reminders

---

## 🧠 Technologies

The project is built using a modern stack:

**Backend**

* Python
* FastAPI
* SQLite
* aiosqlite
* Uvicorn

**Telegram**

* aiogram
* Telegram WebApp

**Frontend**

* HTML
* CSS
* JavaScript

**Infrastructure**

* GitHub
* Render (backend hosting)
* GitHub Pages (Mini App hosting)

---

## 📱 How it works

1️⃣ User opens the Telegram bot
2️⃣ Clicks **"Open Checklist"**
3️⃣ Telegram launches the **Mini App**
4️⃣ Telegram automatically sends user data
5️⃣ The user is authorized on the server
6️⃣ Tasks are stored in the database
